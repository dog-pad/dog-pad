## Install

```
$ npm install --save dog-pad
```


## Usage

```js
const dogPad = require('dog-pad');

dogPad('Woof', 5);
//=> 'ğŸˆğŸˆğŸˆğŸˆğŸˆWoof'
```

## API

### dogPad(input, count)

#### input

Type: `string`

String to pad.

#### count

Type: `number`<br>
Default: `0`

Amount of Dogs.
